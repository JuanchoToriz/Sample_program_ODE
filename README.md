## Sample007 : RKF4(5)法
### テーマ : 二重振り子．ルンゲ・クッタ・フェールブルク4(5)次法．

時間刻み幅制御することに精度良く高速に計算することを目標にしたサンプルプログラム．

1970年台からよく使われる埋込み型ルンゲ・クッタ方の一種のルンゲ・クッタ・フェールブルク法（以下RKF4）のサンプルプログラム．
